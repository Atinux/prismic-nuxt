# Getting Started

1. Install dependency
```bash
npx nuxi@latest module add prismic
```

2. Update Nuxt config

```diff [nuxt.config.ts]
  import { defineNuxtConfig } from 'nuxt'

  export default defineNuxtConfig({
+ 	modules: ['@nuxtjs/prismic'],
+ 	prismic: { endpoint: 'your_repository_id' }
  })
```
